🗄️ Esquema de Base de Datos

Archivo: prisma/migrations/006_add_scraping_tables/migration.sql

Tablas Creadas

1. legal_sources

Configuración de fuentes legales a scrapear

CREATE TABLE legal_sources ( id UUID PRIMARY KEY, source_id VARCHAR(100) UNIQUE, name VARCHAR(255), url TEXT, type VARCHAR(50) CHECK (type IN ('primary', 'secondary', 'tertiary')), priority INTEGER, frequency VARCHAR(50), cron_expression VARCHAR(100), is_active BOOLEAN, last_scraped_at TIMESTAMP, config JSONB );

2. scraped_documents

Documentos scrapeados con contenido completo

CREATE TABLE scraped_documents ( id UUID PRIMARY KEY, source_id UUID REFERENCES legal_sources(id), url TEXT, title TEXT, content TEXT, markdown TEXT, html TEXT, metadata JSONB, hash VARCHAR(64), version INTEGER, status VARCHAR(50), UNIQUE(url, hash) );

3. document_versions

Historial de versiones de documentos

CREATE TABLE document_versions ( id UUID PRIMARY KEY, document_id UUID REFERENCES scraped_documents(id), url TEXT, previous_hash VARCHAR(64), current_hash VARCHAR(64), change_type VARCHAR(50), changes_detected JSONB, version_number INTEGER );

4. scraping_jobs

Historial de ejecución de jobs

CREATE TABLE scraping_jobs ( id UUID PRIMARY KEY, job_id VARCHAR(100), source_id UUID REFERENCES legal_sources(id), status VARCHAR(50), documents_found INTEGER, documents_scraped INTEGER, documents_failed INTEGER, errors JSONB, started_at TIMESTAMP, completed_at TIMESTAMP, duration_ms INTEGER );
✓ Optimizaciones:
  • Índices full-text search en title y content (Spanish)
  • Índices JSONB para queries de metadata
  • Triggers auto-update para updated_at
  • Data inicial de 10 fuentes legales

Herramientas MCP Disponibles

1. firecrawl_map

Mapea un sitio web para descubrir URLs de documentos

const result = await mcp__firecrawl_mcp__firecrawl_map({ url: "https://www.registroficial.gob.ec", search: "*.pdf", limit: 100 }); // Retorna: { links: [ { url, title, description }, ... ] }

2. firecrawl_scrape

Scrapea un documento específico con extracción completa

const result = await mcp__firecrawl_mcp__firecrawl_scrape({ url: documentUrl, formats: ["markdown", "html"], onlyMainContent: true }); // Retorna: { content, markdown, html, links, metadata }

3. firecrawl_extract

Extrae metadatos estructurados mediante AI

const schema = { type: "object", properties: { documentType: { type: "string" }, documentNumber: { type: "string" }, publicationDate: { type: "string" }, title: { type: "string" }, institution: { type: "string" } } }; const result = await mcp__firecrawl_mcp__firecrawl_extract({ urls: [documentUrl], schema: schema });

4. firecrawl_search

Busca documentos legales por query

const result = await mcp__firecrawl_mcp__firecrawl_search({ query: "código civil ecuador", limit: 10, sources: ["web"] });
