✨ Características del Visor PDF

📄 Renderizado Profesional

Utiliza react-pdf y PDF.js para renderizado de alta calidad con capa de texto completa para selección y copiado.

🔍 Controles de Zoom Avanzados

Zoom desde 50% hasta 200%, ajuste automático a ancho/página, controles intuitivos con teclado (+/-) y mouse.

📑 Navegación Inteligente

Navegación por páginas con flechas, salto directo a página específica, atajos de teclado (← →), scroll suave.

✂️ Selección y Copiado

Capa de texto habilitada para seleccionar y copiar texto directamente del PDF a los campos del formulario.

🔄 Rotación de Documento

Rotar el PDF 90° para documentos escaneados en orientación incorrecta. Persistencia del estado de rotación.

⬇️ Descarga Directa

Botón de descarga que genera URL presignada de S3 para download directo y seguro del documento original.

🖨️ Función de Impresión

Impresión optimizada del documento PDF con diálogo nativo del navegador y configuración de impresora.

🔎 Búsqueda de Texto

Barra de búsqueda integrada para encontrar texto específico dentro del PDF con highlight de resultados.

⌨️ Atajos de Teclado

Navegación completa por teclado: flechas (páginas), +/- (zoom), Ctrl+F (buscar), Ctrl+P (imprimir).

💾 Estado Persistente

Mantiene la página actual, nivel de zoom y rotación mientras el usuario edita los metadatos.

⏳ Loading Optimizado

Spinners elegantes y mensajes informativos durante la carga del PDF con indicador de progreso.

📱 Diseño Responsivo

Modal en pantalla completa (95vw x 95vh) con layout adaptativo para diferentes tamaños de pantalla.

⚙️ Características Técnicas Avanzadas

🚀 Lazy Loading

Páginas del PDF se cargan bajo demanda, no todas a la vez. Optimiza memoria y rendimiento.

🎨 Canvas Rendering

Renderizado de alta calidad usando HTML5 Canvas con soporte para dispositivos de alta densidad (Retina).

📝 Text Layer

Capa de texto transparente sobre el canvas que permite selección nativa del navegador.

🔐 Seguridad Multi-Capa

JWT auth + presigned URLs + bucket privado + encriptación en reposo = máxima seguridad.

⚡ Web Workers

PDF.js usa Web Workers para procesar PDFs en background sin bloquear UI principal.

💾 Estado Persistente

Mantiene página actual, zoom y rotación en memoria durante toda la sesión de edición.

🔄 Flujo de Uso del Sistema

Paso Acción del Usuario Respuesta del Sistema
1 Usuario hace clic en "✏️ Editar" en un documento Abre modal full-screen con layout split 50/50
2 Sistema carga automáticamente el PDF Llama a GET /legal-documents-v2/:id/file, obtiene presigned URL, carga PDF en visor
3 Usuario navega por el PDF con controles Renderiza páginas bajo demanda, aplica zoom, permite selección de texto
4 Usuario selecciona texto en el PDF y copia (Ctrl+C) Texto copiado al portapapeles, listo para pegar en formulario
5 Usuario hace clic en "🤖 Extraer con IA" GPT-4 analiza contenido, extrae metadatos, muestra sugerencias en panel lateral
6 Usuario revisa sugerencias de IA vs. PDF visible Permite comparación visual directa, acepta/rechaza cada sugerencia
7 Usuario completa edición y hace clic en "💾 Guardar" Guarda cambios vía PUT /legal-documents-v2/:id, muestra confirmación
8 (Opcional) Usuario descarga o imprime el PDF Genera nueva presigned URL, inicia descarga/impresión del documento original
