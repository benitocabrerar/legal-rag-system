🤖 Sección 1: Uso de Inteligencia Artificial

Esta sección muestra el consumo de recursos de IA durante el procesamiento del documento.

Modelo Metadatos: GPT-4
Modelo utilizado para la extracción automática de metadatos del documento (tipo de norma, fecha de publicación, número de registro oficial, etc.)
Ejemplo: gpt-4
Tokens Metadatos
Número total de tokens consumidos en la extracción de metadatos. Incluye:
  • Prompt Tokens: Tokens enviados al modelo (texto del documento + instrucciones)
  • Completion Tokens: Tokens generados por el modelo (JSON con metadatos)
  • Total Tokens: Suma de ambos
Ejemplo: 1,250 tokens (1,000 prompt + 250 completion)
Costo Metadatos
Costo estimado de la extracción de metadatos basado en precios de OpenAI 2024:
Fórmula:
Costo = (Prompt Tokens × $0.03/1K) + (Completion Tokens × $0.06/1K)
Ejemplo: $0.0450 ($0.03 input + $0.015 output)
Modelo Embeddings: text-embedding-ada-002
Modelo utilizado para generar embeddings (vectores semánticos) de cada chunk del documento para búsqueda RAG.
Ejemplo: text-embedding-ada-002
Tokens Embeddings
Número total de tokens procesados para generar embeddings. Cada chunk de 1000 caracteres consume aproximadamente 250-300 tokens.
Ejemplo: 45,000 tokens (180 chunks × ~250 tokens/chunk)
Costo Embeddings
Costo de generar todos los embeddings del documento:
Fórmula:
Costo = Total Tokens × $0.0001/1K
Ejemplo: $0.0045 (45,000 tokens × $0.0001/1K)
📊 Tokens Totales
Suma de todos los tokens consumidos (metadatos + embeddings).
Ejemplo: 46,250 tokens
💰 Costo Total Estimado
Costo total de procesamiento de IA para este documento. Suma de costos de metadatos y embeddings.
Ejemplo: $0.0495 (~5 centavos de dólar)

🌐 Sección 2: Información de Red

Esta sección muestra detalles de conectividad y red durante la carga.

IP Cliente
Dirección IP desde donde se realizó la carga. Útil para auditoría y seguridad. Se captura usando múltiples métodos de fallback:
  • x-forwarded-for (cuando está detrás de proxy/balanceador)
  • x-real-ip (cuando usa proxy Nginx)
  • socket.remoteAddress (conexión directa)
Ejemplo: 192.168.1.100 o ::1 (localhost IPv6)
IP Base de Datos
Host del servidor PostgreSQL donde se almacenó el documento. Extraído automáticamente de la variable DATABASE_URL.
Ejemplo: dpg-d46iarje5dus73ar46c0-a.oregon-postgres.render.com
Región Servidor
Región geográfica donde está alojado el servidor de base de datos.
Ejemplo: Oregon, USA
Protocolo de Conexión
Protocolo usado para conectarse a la base de datos. PostgreSQL con cifrado SSL para seguridad.
Ejemplo: PostgreSQL SSL
Protocolo de Carga
Protocolos usados durante la carga del archivo:
  • HTTP: Para la carga inicial de chunks
  • WebSocket: Para actualizaciones en tiempo real del progreso
Ejemplo: HTTP/WebSocket

💻 Sección 3: Información del Sistema

Esta sección muestra el estado del servidor al momento de la carga.

Versión Node.js
Versión del runtime de Node.js ejecutando el servidor. Importante para compatibilidad y debugging.
Ejemplo: v18.17.0 o v20.10.0
Plataforma
Sistema operativo del servidor. Valores posibles:
  • win32 - Windows
  • linux - Linux
  • darwin - macOS
Ejemplo: win32
Memoria Usada
Memoria heap actualmente en uso por el proceso de Node.js. Útil para monitorear el consumo de recursos.
Ejemplo: 125.50 MB
Memoria Total
Tamaño total del heap asignado al proceso de Node.js.
Ejemplo: 256.00 MB
Uptime del Servidor
Tiempo que el servidor ha estado ejecutándose sin reiniciar. Indica estabilidad del servicio.
Ejemplo: 45.3 minutos

🧮 Ejemplo de Cálculo Completo

Escenario: Documento de 200 páginas

1. Extracción de Metadatos con GPT-4
- Prompt Tokens: 1,000 tokens
- Completion Tokens: 250 tokens
- Costo = (1,000 × $0.03/1K) + (250 × $0.06/1K) = $0.03 + $0.015 = $0.045
2. Generación de Embeddings
- Texto total: ~100,000 palabras ≈ 400,000 caracteres
- Chunks generados: 400 chunks (1,000 caracteres cada uno)
- Tokens embeddings: 400 × 250 = 100,000 tokens
- Costo = 100,000 × $0.0001/1K = $0.01
3. Costo Total
$0.045 (metadatos) + $0.01 (embeddings) = $0.055 (~6 centavos)
💡 Conclusión: Procesar un documento legal de 200 páginas cuesta aproximadamente 6 centavos de dólar.

📋 Casos de Uso de las Métricas

1. Control de Costos

Monitorear el costo acumulado de procesamiento de documentos para presupuestar recursos de IA.

2. Auditoría de Seguridad

Rastrear las IPs de carga para detectar accesos no autorizados o actividad sospechosa.

3. Optimización de Rendimiento

Analizar el uso de memoria y uptime para identificar necesidades de escalamiento.

4. Depuración Técnica

Usar versión de Node.js, plataforma y detalles de red para resolver problemas de compatibilidad.

5. Reporting Ejecutivo

Generar reportes de uso de IA para stakeholders y justificación de inversión en tecnología.
