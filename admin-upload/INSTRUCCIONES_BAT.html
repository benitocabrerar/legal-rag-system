🚀 Archivos BAT - Servidor de Carga

Gestiona el servidor con un simple doble clic

▶️ start-server.bat
Función: Inicia el servidor de carga de archivos grandes
📖 Cómo usar:
  • 1 Haz doble clic en el archivo
  • 2 Se abrirá una ventana de terminal negra
  • 3 Espera a ver: Servidor iniciado en: http://localhost:3333
  • 4 Abre tu navegador en: http://localhost:3333
✅ Qué hace automáticamente:
  • Verifica que Node.js esté instalado
  • Instala dependencias si faltan (npm install)
  • Inicia el servidor en puerto 3333
  • Mantiene la ventana abierta para ver logs
⏹️ stop-server.bat
Función: Detiene el servidor de carga
📖 Cómo usar:
  • 1 Haz doble clic en el archivo
  • 2 Se abrirá una ventana que dice "Servidor detenido"
  • 3 La ventana se cerrará automáticamente después de 3 segundos
✅ Qué hace automáticamente:
  • Busca procesos usando el puerto 3333
  • Busca procesos de Node.js ejecutando server.js
  • Detiene todos los procesos encontrados
  • Confirma que el servidor se detuvo correctamente
🔄 restart-server.bat
Función: Detiene y reinicia el servidor (útil después de cambios en código)
📖 Cómo usar:
  • 1 Haz doble clic en el archivo
  • 2 Verás que primero detiene el servidor
  • 3 Espera 2 segundos
  • 4 Inicia el servidor nuevamente
✅ Cuándo usar:
  • Después de modificar archivos de código
  • Después de cambiar variables de entorno (.env)
  • Si el servidor se queda "colgado"
  • Para aplicar cambios en la configuración
⚠️ Notas Importantes:
  • NO cierres la ventana negra mientras el servidor esté funcionando
  • Si cierras la ventana, el servidor se detendrá automáticamente
  • Para detener el servidor correctamente, usa stop-server.bat
  • También puedes presionar Ctrl+C en la ventana del servidor
🛠️ Solución de Problemas
❌ Error: "Node.js no está instalado"
  • Descarga Node.js desde: https://nodejs.org
  • Instala la versión LTS (recomendada)
  • Reinicia tu computadora
  • Intenta nuevamente
❌ Error: "Puerto 3333 ya está en uso"
  • Ejecuta stop-server.bat
  • Espera 5 segundos
  • Ejecuta start-server.bat nuevamente
❌ La página no carga (ERR_CONNECTION_REFUSED)
  • Verifica que la ventana del servidor esté abierta
  • Verifica que diga "Servidor iniciado en: http://localhost:3333"
  • Verifica que no haya errores en rojo en la ventana
  • Intenta con restart-server.bat

✅ Flujo Normal de Trabajo

  1. Inicio del día: Doble clic en start-server.bat
  2. Trabajar: Abrir http://localhost:3333 y cargar documentos
  3. Si haces cambios: Usar restart-server.bat
  4. Fin del día: Doble clic en stop-server.bat
