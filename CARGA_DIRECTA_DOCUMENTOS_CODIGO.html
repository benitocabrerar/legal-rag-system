📖 Introducción

¿Qué es la Carga Directa Mediante Código?

La carga directa es un método alternativo que permite a los administradores subir documentos legales sin pasar por la interfaz web, ejecutando scripts directamente en el servidor o localmente con conexión a la base de datos.

Escenario Problemático Resuelto

🎯 Problema Actual

Un administrador tiene un archivo PDF de 200 MB (1000 páginas) del Código Orgánico Integral Penal que necesita subir al sistema.

❌ Limitaciones Actuales vía Web UI:

  • Límite frontend: 50 MB
  • Límite Fastify multipart: configurable pero riesgoso
  • Timeout HTTP: 60 segundos
  • RAM Render Free: 256 MB (OOM risk)
  • Ancho de banda: upload lento desde ubicación remota

✅ Solución con Carga Directa:

  • Sin límites de tamaño HTTP
  • Sin timeouts de request
  • Procesamiento optimizado (streaming, batch)
  • Acceso directo a filesystem del servidor
  • Control total del flujo de procesamiento

Tecnologías Involucradas

Node.js / Bun Prisma CLI SSH / Render Shell fs / fs-extra pdf-lib SFTP / SCP BullMQ Commander.js

Método 1: Script Local con Conexión Remota a BD

MÁS SIMPLE Desarrollo Local

Descripción

Ejecutar un script Node.js desde tu máquina local que se conecta directamente a la base de datos PostgreSQL en Render.

Requisitos

  • DATABASE_URL configurado en archivo .env.local
  • Archivo PDF en filesystem local
  • Node.js instalado
  • Dependencias: @prisma/client, pdf.js-extract, openai

Código Completo - upload-document.js

// scripts/upload-document.js
import { PrismaClient } from '@prisma/client';
import { readFileSync } from 'fs';
import { PDFExtract } from 'pdf.js-extract';
import OpenAI from 'openai';
import path from 'path';

const prisma = new PrismaClient();
const pdfExtract = new PDFExtract();
const openai = new OpenAI({ apiKey: process.env.OPENAI_API_KEY });

async function uploadLargeDocument(filePath, metadata) {
  console.log(`📄 Procesando: ${path.basename(filePath)}`);

  // 1. Leer archivo
  const pdfBuffer = readFileSync(filePath);
  const fileSizeMB = pdfBuffer.length / (1024 * 1024);
  console.log(`📊 Tamaño: ${fileSizeMB.toFixed(2)} MB`);

  // 2. Extraer texto
  console.log('🔍 Extrayendo texto del PDF...');
  const data = await pdfExtract.extractBuffer(pdfBuffer);
  const fullText = data.pages
    .map(page => page.content.map(item => item.str).join(' '))
    .join('\n\n');

  console.log(`📝 Páginas extraídas: ${data.pages.length}`);
  console.log(`📏 Caracteres totales: ${fullText.length}`);

  // 3. Crear documento
  console.log('💾 Creando registro en base de datos...');
  const document = await prisma.legalDocument.create({
    data: {
      normType: metadata.normType,
      normTitle: metadata.normTitle,
      legalHierarchy: metadata.legalHierarchy,
      publicationType: metadata.publicationType,
      publicationNumber: metadata.publicationNumber,
      publicationDate: metadata.publicationDate ? new Date(metadata.publicationDate) : null,
      jurisdiction: metadata.jurisdiction || 'NACIONAL',
      documentState: 'ORIGINAL',
      content: fullText,
      uploadedBy: metadata.adminUserId,
      metadata: {
        fileSizeMB,
        totalPages: data.pages.length,
        uploadMethod: 'direct_script',
        uploadedAt: new Date().toISOString()
      }
    }
  });

  console.log(`✅ Documento creado: ${document.id}`);

  // 4. Generar chunks
  console.log('✂️  Dividiendo en chunks...');
  const chunkSize = 1000;
  const chunks = [];

  for (let i = 0; i < fullText.length; i += chunkSize) {
    chunks.push(fullText.slice(i, i + chunkSize));
  }

  console.log(`📦 Total chunks: ${chunks.length}`);

  // 5. Generar embeddings y guardar chunks
  console.log('🧠 Generando embeddings...');
  let processedChunks = 0;

  for (let i = 0; i < chunks.length; i++) {
    try {
      const embeddingResponse = await openai.embeddings.create({
        model: 'text-embedding-ada-002',
        input: chunks[i],
      });

      const embedding = embeddingResponse.data[0].embedding;

      await prisma.legalDocumentChunk.create({
        data: {
          legalDocumentId: document.id,
          content: chunks[i],
          chunkIndex: i,
          embedding: embedding
        }
      });

      processedChunks++;

      // Progreso cada 50 chunks
      if (processedChunks % 50 === 0) {
        console.log(`   ⏳ Procesados ${processedChunks}/${chunks.length} chunks...`);
      }

    } catch (error) {
      console.error(`   ⚠️  Error en chunk ${i}:`, error.message);
      // Continuar con el siguiente chunk
    }
  }

  console.log(`\n✅ Procesamiento completado!`);
  console.log(`📊 Estadísticas:`);
  console.log(`   - Documento ID: ${document.id}`);
  console.log(`   - Páginas: ${data.pages.length}`);
  console.log(`   - Chunks procesados: ${processedChunks}/${chunks.length}`);
  console.log(`   - Tamaño archivo: ${fileSizeMB.toFixed(2)} MB`);
  console.log(`   - Costo estimado OpenAI: $${(processedChunks * 0.0001).toFixed(4)}`);

  await prisma.$disconnect();
  return document;
}

// Uso
const metadata = {
  normType: 'ORGANIC_CODE',
  normTitle: 'Código Orgánico Integral Penal',
  legalHierarchy: 'CODIGOS_ORGANICOS',
  publicationType: 'ORDINARIO',
  publicationNumber: 'Suplemento 180',
  publicationDate: '2014-02-10',
  jurisdiction: 'NACIONAL',
  adminUserId: 'USER_ID_AQUI' // ID del admin que sube
};

uploadLargeDocument('./documentos/coip-2024.pdf', metadata)
  .then(() => console.log('\n🎉 Carga completada con éxito!'))
  .catch(error => {
    console.error('\n❌ Error durante la carga:', error);
    process.exit(1);
  });

Método 3: CLI Interactiva

UX Mejorada Commander.js

Descripción

#!/usr/bin/env node
// scripts/legal-cli.js

import { Command } from 'commander';
import inquirer from 'inquirer';
import chalk from 'chalk';
import ora from 'ora';
import { PrismaClient } from '@prisma/client';
import { uploadLargeDocument } from './upload-document.js';

const program = new Command();
const prisma = new PrismaClient();

program
  .name('legal-cli')
  .description('Herramienta CLI para cargar documentos legales grandes')
  .version('1.0.0');

program
  .command('upload')
  .description('Subir un documento legal grande')
  .option('-f, --file <path>', 'Ruta al archivo PDF')
  .option('-i, --interactive', 'Modo interactivo')
  .action(async (options) => {
    console.log(chalk.blue.bold('\n📄 Legal Document Uploader CLI\n'));

    let filePath = options.file;
    let metadata = {};

    // Modo interactivo
    if (options.interactive || !filePath) {
      const answers = await inquirer.prompt([
        {
          type: 'input',
          name: 'filePath',
          message: '📂 Ruta al archivo PDF:',
          default: './documentos/ejemplo.pdf',
          validate: (input) => {
            if (!input.endsWith('.pdf')) {
              return 'Debe ser un archivo PDF';
            }
            return true;
          }
        },
        {
          type: 'list',
          name: 'normType',
          message: '📋 Tipo de Norma:',
          choices: [
            'CONSTITUTIONAL_NORM',
            'ORGANIC_LAW',
            'ORDINARY_LAW',
            'ORGANIC_CODE',
            'ORDINARY_CODE',
            'REGULATION_GENERAL'
          ]
        },
        {
          type: 'input',
          name: 'normTitle',
          message: '📝 Título de la Norma:',
          validate: (input) => input.length > 0
        },
        {
          type: 'list',
          name: 'legalHierarchy',
          message: '⚖️  Jerarquía Legal:',
          choices: [
            'CONSTITUCION',
            'LEYES_ORGANICAS',
            'LEYES_ORDINARIAS',
            'CODIGOS_ORGANICOS',
            'CODIGOS_ORDINARIOS',
            'REGLAMENTOS'
          ]
        },
        {
          type: 'input',
          name: 'publicationNumber',
          message: '📰 Número de Publicación:',
        },
        {
          type: 'confirm',
          name: 'confirm',
          message: '¿Confirmas la carga de este documento?',
          default: false
        }
      ]);

      if (!answers.confirm) {
        console.log(chalk.yellow('\n❌ Carga cancelada por el usuario\n'));
        process.exit(0);
      }

      filePath = answers.filePath;
      metadata = answers;
    }

    // Obtener admin user ID
    const spinner = ora('Buscando usuario administrador...').start();
    const admin = await prisma.user.findFirst({
      where: { role: 'admin' }
    });

    if (!admin) {
      spinner.fail('No se encontró usuario administrador');
      process.exit(1);
    }

    spinner.succeed(`Admin encontrado: ${admin.email}`);
    metadata.adminUserId = admin.id;

    // Procesar documento
    const uploadSpinner = ora('Procesando documento...').start();

    try {
      const document = await uploadLargeDocument(filePath, metadata);
      uploadSpinner.succeed(chalk.green(`✅ Documento cargado: ${document.id}`));
    } catch (error) {
      uploadSpinner.fail(chalk.red(`❌ Error: ${error.message}`));
      process.exit(1);
    }

    await prisma.$disconnect();
  });

// scripts/batch-upload.js
import { readFileSync } from 'fs';
import { uploadLargeDocument } from './upload-document.js';
import chalk from 'chalk';

async function batchUpload(configPath) {
  console.log(chalk.blue.bold('\n📦 Batch Upload - Carga Masiva de Documentos\n'));

  const config = JSON.parse(readFileSync(configPath, 'utf-8'));
  console.log(`📋 Total documentos a procesar: ${config.length}\n`);

  const results = {
    success: [],
    failed: []
  };

  for (let i = 0; i < config.length; i++) {
    const item = config[i];
    console.log(chalk.cyan(`\n[${i + 1}/${config.length}] Procesando: ${item.metadata.normTitle}`));
    console.log(chalk.gray(`   Archivo: ${item.filePath}`));

    try {
      const document = await uploadLargeDocument(item.filePath, item.metadata);

      results.success.push({
        title: item.metadata.normTitle,
        id: document.id,
        file: item.filePath
      });

      console.log(chalk.green(`   ✅ Completado: ${document.id}`));
    } catch (error) {
      results.failed.push({
        title: item.metadata.normTitle,
        file: item.filePath,
        error: error.message
      });

      console.log(chalk.red(`   ❌ Error: ${error.message}`));
    }

    // Pausa entre documentos para no saturar OpenAI API
    if (i < config.length - 1) {
      console.log(chalk.yellow('   ⏳ Esperando 5 segundos...'));
      await new Promise(resolve => setTimeout(resolve, 5000));
    }
  }

  // Resumen final
  console.log(chalk.blue.bold('\n\n📊 RESUMEN DE CARGA MASIVA\n'));
  console.log(chalk.green(`✅ Exitosos: ${results.success.length}`));
  console.log(chalk.red(`❌ Fallidos: ${results.failed.length}`));

  if (results.success.length > 0) {
    console.log(chalk.green('\n✅ Documentos cargados exitosamente:'));
    results.success.forEach((item, idx) => {
      console.log(`   ${idx + 1}. ${item.title} (${item.id})`);
    });
  }

  if (results.failed.length > 0) {
    console.log(chalk.red('\n❌ Documentos fallidos:'));
    results.failed.forEach((item, idx) => {
      console.log(`   ${idx + 1}. ${item.title}`);
      console.log(`      Error: ${item.error}`);
    });
  }

  console.log('\n');
}

// Ejecutar
const configPath = process.argv[2] || './documents-batch.json';
batchUpload(configPath);

Método 5: API Interna Protegida

Híbrido API + Script

Descripción

Crear un endpoint API especial protegido por API key que permita subir archivos grandes sin los límites normales.

Endpoint Especial - /admin/upload-large

// src/routes/admin/upload-large.ts

import { FastifyInstance } from 'fastify';
import { createWriteStream } from 'fs';
import { pipeline } from 'stream/promises';
import { randomUUID } from 'crypto';

export async function adminUploadRoutes(fastify: FastifyInstance) {

  // Endpoint especial sin límites
  fastify.post('/admin/upload-large', {
    preHandler: [
      fastify.authenticate,
      async (request, reply) => {
        const user = request.user as any;

        // Solo admins o API key especial
        if (user.role !== 'admin' && !request.headers['x-admin-api-key']) {
          return reply.code(403).send({ error: 'Forbidden' });
        }

        // Validar API key si se proporciona
        if (request.headers['x-admin-api-key']) {
          const validKey = process.env.ADMIN_API_KEY;
          if (request.headers['x-admin-api-key'] !== validKey) {
            return reply.code(401).send({ error: 'Invalid API key' });
          }
        }
      }
    ]
  }, async (request, reply) => {

    // Sin límites de tamaño
    const data = await request.file({
      limits: {
        fileSize: Infinity, // Sin límite
      }
    });

    if (!data) {
      return reply.code(400).send({ error: 'No file provided' });
    }

    const tempPath = `/tmp/${randomUUID()}.pdf`;

    // Guardar temporalmente con streaming
    await pipeline(
      data.file,
      createWriteStream(tempPath)
    );

    // Procesar en background (BullMQ)
    const job = await documentQueue.add('process-large-document', {
      filePath: tempPath,
      metadata: JSON.parse(data.fields.metadata?.value || '{}'),
      userId: (request.user as any)?.id
    });

    return reply.send({
      success: true,
      message: 'Document queued for processing',
      jobId: job.id,
      statusUrl: `/admin/job-status/${job.id}`
    });
  });

  // Check job status
  fastify.get('/admin/job-status/:jobId', {
    preHandler: [fastify.authenticate]
  }, async (request, reply) => {
    const { jobId } = request.params as any;
    const job = await documentQueue.getJob(jobId);

    if (!job) {
      return reply.code(404).send({ error: 'Job not found' });
    }

    const state = await job.getState();
    const progress = job.progress;

    return reply.send({
      jobId,
      state,
      progress,
      result: await job.returnvalue
    });
  });
}

Cliente para el endpoint - upload-via-api.js

// scripts/upload-via-api.js
import FormData from 'form-data';
import { createReadStream } from 'fs';
import fetch from 'node-fetch';

async function uploadViaAPI(filePath, metadata) {
  const form = new FormData();

  form.append('file', createReadStream(filePath));
  form.append('metadata', JSON.stringify(metadata));

  const response = await fetch('https://legal-rag-backend.onrender.com/admin/upload-large', {
    method: 'POST',
    headers: {
      'x-admin-api-key': process.env.ADMIN_API_KEY
    },
    body: form
  });

  const result = await response.json();
  console.log('✅ Carga iniciada:', result);

  // Polling para ver progreso
  if (result.jobId) {
    let completed = false;

    while (!completed) {
      await new Promise(r => setTimeout(r, 5000)); // 5 segundos

      const statusResponse = await fetch(result.statusUrl, {
        headers: {
          'Authorization': `Bearer ${process.env.AUTH_TOKEN}`
        }
      });

      const status = await statusResponse.json();
      console.log(`⏳ Estado: ${status.state}, Progreso: ${status.progress}%`);

      if (status.state === 'completed') {
        completed = true;
        console.log('✅ Procesamiento completado!');
      } else if (status.state === 'failed') {
        completed = true;
        console.error('❌ Procesamiento fallido');
      }
    }
  }
}

Ejemplo de Flujo Completo

Escenario Real: Migración de Base Legal Completa

Contexto: Tienes 20 códigos legales (promedio 150 MB cada uno) que necesitas migrar.

Paso a Paso:

  1. Preparación:
    • Descargar todos los PDFs a ./documentos/
    • Crear documents-batch.json con metadata
  2. Validación: 
    node scripts/legal-cli.js validate-batch documents-batch.json
  3. Carga (durante la noche): 
    nohup node scripts/batch-upload.js documents-batch.json > upload.log 2>&1 &
  4. Monitoreo: 
    tail -f upload.log
  5. Verificación al día siguiente: 
    node scripts/legal-cli.js list -l 25

Tiempo estimado: 6-8 horas para 20 documentos (procesamiento overnight)

Costo OpenAI: ~$3-5 USD (20 docs × ~$0.20 cada uno)

Cuándo Usar Cada Método

✅ Usar Web UI cuando:

  • El archivo es <50 MB
  • Usuario final (no técnico) sube el documento
  • Necesitas auditoría automática (quien subió, cuándo)
  • El documento no es urgente
  • Quieres simplicidad máxima

⚠️ Usar Carga Directa cuando:

  • El archivo es >50 MB
  • Necesitas subir múltiples documentos (migración)
  • Timeout HTTP es un problema
  • Quieres control total del procesamiento
  • Necesitas debugging detallado
  • Procesamiento durante horarios no hábiles (overnight)

🔒 Consideraciones de Seguridad

⚠️ Riesgos de la Carga Directa

1. Exposición de DATABASE_URL

Si ejecutas scripts localmente, la DATABASE_URL debe estar en tu .env.local. Esto representa un riesgo si:

  • El archivo .env se commitea al repositorio (usar .gitignore)
  • La máquina local es comprometida
  • Múltiples personas tienen acceso

Mitigación:

  • ✅ Usar roles de PostgreSQL con permisos limitados
  • ✅ Rotar DATABASE_URL periódicamente
  • ✅ Usar IP whitelisting en Render
  • ✅ Encriptar .env con herramientas como dotenv-vault

import { readFileSync } from 'fs';

// Verificar magic bytes (PDF signature)
function validatePDF(filePath) {
  const buffer = readFileSync(filePath);
  const signature = buffer.toString('hex', 0, 4);

  // PDF signature: %PDF (25 50 44 46)
  if (signature !== '25504446') {
    throw new Error('Archivo no es un PDF válido');
  }
}

3. Rate Limiting

// Limitar llamadas a OpenAI API
const rateLimiter = {
  calls: 0,
  resetTime: Date.now() + 60000, // 1 minuto

  async wait() {
    if (this.calls >= 60) { // 60 calls/min
      const waitTime = this.resetTime - Date.now();
      if (waitTime > 0) {
        await new Promise(r => setTimeout(r, waitTime));
      }
      this.calls = 0;
      this.resetTime = Date.now() + 60000;
    }
    this.calls++;
  }
};
