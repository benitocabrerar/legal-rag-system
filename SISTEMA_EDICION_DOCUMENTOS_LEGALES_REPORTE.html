🔄 Flujo de Trabajo del Usuario

1. Usuario hace clic en botón "✏️ Editar"

Se abre el modal de edición con todos los campos pre-poblados con los valores actuales del documento seleccionado.

2. Usuario puede editar campos manualmente

Todos los campos son editables: título, tipo de norma, jerarquía, publicación, fechas, estado, jurisdicción.

3. Usuario hace clic en "🤖 Extraer Metadatos con IA"

Se envía el contenido del documento a GPT-4 Turbo. Muestra spinner mientras procesa (5-10 segundos).

4. IA retorna sugerencias con confianza

Panel lateral muestra cada campo sugerido con: valor, badge de confianza (Alta/Media/Baja) y razonamiento.

5. Usuario revisa y acepta sugerencias selectivamente

Para cada sugerencia, el usuario puede hacer clic en "Aceptar Sugerencia" para aplicarla al formulario, o ignorarla.

6. Usuario modifica valores si es necesario

Después de aceptar sugerencias, el usuario puede ajustar manualmente cualquier campo antes de guardar.

7. Usuario hace clic en "💾 Guardar Cambios"

Los cambios se guardan vía API PUT. Muestra alerta de éxito/error. Recarga la lista de documentos.

8. (Opcional) Usuario regenera embeddings

Si el documento tiene chunks < 1, aparece botón "🔄 Regenerar Embeddings". Al hacer clic, se confirma la acción y se regeneran todos los chunks y embeddings.

⚠️ Manejo de Errores

Estrategias Implementadas

Retry con Exponential Backoff

OpenAI API calls usan retry automático: 1s, 2s, 4s de delay para manejar rate limits.

Fallback de IA

Si GPT-4 falla, retorna valores por defecto con confianza "low" y mensaje de error claro.

Validación de Datos

Content mínimo de 100 caracteres para extracción IA. Campos requeridos validados en frontend y backend.

Mensajes Específicos

Cada tipo de error retorna mensaje específico: 404 Not Found, 400 Bad Request, 403 Forbidden, 500 Internal Error.

Logging Detallado

Console.log en desarrollo con emojis y timestamps. Fastify.log en producción.

Estado de UI

Spinners mientras se procesa. Alertas detalladas de éxito/error. Disable buttons durante operaciones.
