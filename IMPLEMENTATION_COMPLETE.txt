╔══════════════════════════════════════════════════════════════════════════════╗
║                             TECHNICAL DETAILS                                ║
╚══════════════════════════════════════════════════════════════════════════════╝

UPLOAD FLOW:
  User → Frontend → Backend → PostgreSQL → S3
  ↓
  Metadata stored: {s3Key, s3Bucket, fileSize, originalFilename}

DOWNLOAD FLOW:
  User → GET /file → Generate Presigned URL → 302 Redirect → S3

DELETE FLOW:
  User → DELETE → Soft Delete DB → Delete from S3 → Audit Log

FILE STRUCTURE IN S3:
  legal-documents/
    └─ {documentId}/
       └─ {timestamp}_{filename}.pdf

PRESIGNED URL:
  ├─ Expiration: 1 hour
  ├─ No AWS credentials exposed
  ├─ Generated per request
  └─ Secure temporary access
